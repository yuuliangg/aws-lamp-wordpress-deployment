## Full Project Entry
AWS Cloud Hosting & Scalable WordPress Deployment  
AWS EC2 • Amazon Linux • Apache • MariaDB • PHP • WordPress • RDS • EFS • Auto Scaling

- Deployed an Amazon Linux EC2 instance as a cloud-hosted WordPress environment, configuring security groups, Elastic IP association, SSH access, Apache, MariaDB, PHP, and HTTP connectivity.
- Enhanced the initial single-instance deployment with Route 53, an Application Load Balancer, and Auto Scaling to improve fault tolerance and support automatic instance replacement under failure or load.
- Integrated Amazon RDS and EFS concepts to reduce database administration workload and support shared WordPress upload storage across multiple instances.
- Validated the deployment through browser testing, service checks, local database stop testing, EFS upload checks, and ApacheBench load testing.
